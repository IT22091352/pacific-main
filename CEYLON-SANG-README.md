# Ceylon Sang - Sri Lankan Tour Guide Website

## Project Overview
A complete, professional tour guide website for "Ceylon Sang" - a Sri Lankan tour guide service. The website features ocean blue (#006994) and sunset orange (#FF6B35) as the primary color scheme.

## Pages Created

### 1. **index.html** - Homepage
- Hero section with booking form
- Welcome section with services overview
- Popular destinations preview (3 featured destinations)
- Tour packages preview (3 featured packages)
- Customer testimonials/reviews carousel
- Call-to-action section

### 2. **about.html** - About Page
- Company introduction and story
- Core values section (6 value cards)
- Mission and vision statements
- Statistics counter (years of experience, happy travelers, destinations, satisfaction rate)
- Call-to-action

### 3. **services.html** - Services Page
- Comprehensive list of 6 main services:
  1. Guided Cultural Tours
  2. Wildlife Safari Adventures
  3. Beach & Coastal Experiences
  4. Adventure & Trekking
  5. Tea Plantation Tours
  6. Customized Private Tours
- Additional services section (8 extra services)
- Each service includes detailed descriptions and features

### 4. **destinations.html** - Destinations Page
- Detailed information about 9 popular Sri Lankan destinations:
  1. Sigiriya Rock Fortress
  2. Ella & Nine Arch Bridge
  3. Galle Fort
  4. Yala National Park
  5. Kandy - The Cultural Capital
  6. Mirissa Beach
  7. Nuwara Eliya - Little England
  8. Anuradhapura - Ancient Capital
  9. Polonnaruwa - Medieval Capital
- Each destination includes:
  - Large featured image
  - Location information
  - Highlight badges (UNESCO, Cultural Site, etc.)
  - Detailed description
  - Key highlights list
  - Best time to visit
  - Duration and activities

### 5. **packages.html** - Tour Packages Page
- 9 different tour packages:
  1. Cultural Triangle Explorer (7 days)
  2. Beach & Wildlife Safari (5 days)
  3. Complete Sri Lanka (10 days)
  4. Hill Country & Tea Trails (6 days)
  5. Wildlife & Nature Adventure (4 days)
  6. Coastal Paradise Tour (8 days)
  7. Weekend Getaway (3 days)
  8. Grand Sri Lanka Tour (12 days)
  9. Custom Tour Package (flexible)
- Each package includes:
  - Duration
  - Description
  - Price
  - Key inclusions
  - Book now button
- "What's Included" section showing all package inclusions

### 6. **gallery.html** - Photo Gallery Page
- Photo gallery with filtering by category:
  - All Photos
  - Cultural Sites
  - Wildlife
  - Beaches
  - Nature
  - With Tourists
- 15+ gallery images with captions
- Hover effects on images
- Photo upload form for tourists to share their travel photos:
  - Name and email fields
  - Tour package selection
  - Multiple photo upload (max 10 photos, 5MB each)
  - Caption/description
  - Consent checkbox

### 7. **contact.html** - Contact & Reviews Page
- Contact information section:
  - Address
  - Phone numbers
  - Email addresses
- Contact form with:
  - Name, email, phone
  - Subject
  - Tour package selection
  - Message
- Embedded Google Map
- Customer reviews section (6 detailed reviews)
- Leave a review form with:
  - Name, email, country
  - Tour package selection
  - 5-star rating system
  - Review text
- Social media links section
